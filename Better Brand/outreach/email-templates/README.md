# Email Templates

Outreach email templates for Instantly campaigns. Each template goes through the review queue before first use.

## Template Naming
`[SEQUENCE]-[STEP]-[VERSION].md`
- Example: `cold-website-redesign-step1-v1.md`

## Sequence Types (planned)
1. **cold-website-redesign** — Targeting businesses with outdated websites
2. **cold-no-website** — Targeting businesses with no web presence
3. **cold-local-business** — Targeting local businesses in specific areas
4. **warm-followup** — Follow-up after initial engagement
5. **referral-ask** — Post-delivery referral request

_Templates will be built during Phase 7 (Prospecting & Client Acquisition)._
